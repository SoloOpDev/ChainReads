# ChainReads

Web3 engagement platform with points system. Users read crypto content, make price predictions, and exchange earned points for tokens via smart contract on Base.

## Stack

React, TypeScript, Node.js, PostgreSQL, Solidity, Base Network

## Setup

```bash
npm install
cp .env.example .env
npm run db:push
npm run dev
```

See `GITHUB_SECRETS.md` for deployment variables.
