# GitHub Secrets Setup

## Required GitHub Secrets

Add these secrets to your GitHub repository:
**Settings → Secrets and variables → Actions → New repository secret**

### 1. Telegram API Credentials
- `TELEGRAM_API_ID` - Your Telegram API ID from https://my.telegram.org
- `TELEGRAM_API_HASH` - Your Telegram API Hash from https://my.telegram.org
- `TELEGRAM_SESSION` - Session string generated by `generate_session.py`

### 2. Telegram Channels
- `TELEGRAM_TRADING_CHANNELS` - Comma-separated list (e.g., `cryptosignals,Coin_Signals`)
- `TELEGRAM_AIRDROP_CHANNELS` - Comma-separated list (e.g., `airdropfind,AirdropAnalyst`)

### 3. ImageKit CDN (Free Tier: 20GB storage + 20GB bandwidth)
- `IMAGEKIT_PRIVATE_KEY` - Your ImageKit private API key
- `IMAGEKIT_URL_ENDPOINT` - Your ImageKit URL endpoint (e.g., `https://ik.imagekit.io/your_id`)

**Get ImageKit credentials:**
1. Sign up at https://imagekit.io (free tier)
2. Go to Developer Options → API Keys
3. Copy Private Key and URL Endpoint

### 4. Railway Deployment
- `RAILWAY_API_URL` - Your Railway app URL (e.g., `https://chainreads-production.up.railway.app`)
- `TELEGRAM_UPDATE_SECRET` - Random secret for API authentication (generate with `openssl rand -hex 32`)

## How to Add Secrets

```bash
# Go to your GitHub repository
# Settings → Secrets and variables → Actions → New repository secret
# Add each secret one by one
```

## Verify Secrets

Run the verification script:
```bash
bash scripts/verify-github-secrets.sh
```
